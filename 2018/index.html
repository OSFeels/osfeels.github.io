About

Open Source & Feelings returns for another year! OS&F is a single-track conference highlighting the people behind the code and their experience.

Code of Conduct:
The Short Version

Open Source & Feelings is dedicated to a safe, inclusive, welcoming, and harassment-free conference experience for everyone.

All attendees, speakers, sponsors and volunteers at Open Source Feelings are required to abide by the code of conduct.
